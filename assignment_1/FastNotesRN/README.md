# FastNotes React Native (Expo)

A simple note-taking app built with React Native and Expo for IKT205.

## Sourcecode

Sourcecode is available at: https://github.com/andreaslh200/ikt205-andreaslh

## Features

- Create notes with a title and content
- View all notes in a list
- Tap a note to see its details
- Notes stored in-memory

## Getting Started

```bash
npm install
npx expo start
```

Press `a` to open on an Android emulator, or scan the QR code with the Expo Go app on your phone.

## Tech Stack

- React Native with Expo
- TypeScript
- React Navigation (native stack)
