# FastNotes Kotlin (Jetpack Compose)

A simple note-taking app built with Jetpack Compose for IKT205.

## Sourcecode

Sourcecode is available at: https://github.com/andreaslh200/ikt205-andreaslh

## Features

- Create notes with a title and content
- View all notes in a list
- Tap a note to see its details
- Notes stored in-memory

## Getting Started

1. Open this folder in Android Studio
2. Let Gradle sync finish
3. Run on an emulator or connected device

## Tech Stack

- Kotlin
- Jetpack Compose with Material 3
- Navigation Compose
